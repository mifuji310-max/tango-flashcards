# 暗記シート単語帳 — プロジェクトまとめ

子供のスマホで使う、受験用の英単語・イディオム学習アプリ。ここまでの検討内容と成果物をまとめる。

---

## 1. 作るもの

- **目的**: 受験用の英単語・イディオムを、軽いゲーム感覚で覚える
- **対象**: 子供（英検4級〜1級レベルに対応）
- **端末**: 子供のスマホ（iPhone / Android 両対応）
- **方針**: 過度なゲーム化はしない。バトルモード等は不要。赤シート＋軽い仕分けで淡々と回せるものにする

---

## 2. 技術的アプローチ

**PWA（Webアプリ）を採用。** HTMLファイル1つで完結する形。

採用理由:

- iPhone / Android どちらでも1つで動く
- App Store 審査が不要。ホーム画面に追加すればアプリのように使える
- 単語の追加・修正がすぐ反映できる
- まず自分の子供に使わせるには十分

本格配布・収益化まで考えるなら Flutter や React Native が候補になるが、今回の用途ではPWAで過不足なし。

---

## 3. コンテンツの方針（著作権の整理）

### 却下した案
市販単語帳や既存学習サイト（例: 英語漬け.com）からのコピー・スクレイピングは**しない**。

- 単語帳やサイトのコンテンツには著作権がある。個人利用を超えると侵害にあたる
- 単語の「選定・配列」自体にも編集著作権が認められることがありグレー
- 技術的にも、対象サイトはJSで動的に単語を読み込む作りで単純コピーでは取得できない

### 採用した案（クリーンな構成）
- **どの語を選ぶか** → **NAWL（New Academic Word List）** を参考。NAWLは CC BY-SA ライセンスで、出典明記で自由に利用・改変できる
- **和訳・イディオム** → **すべてオリジナルで作成**。既存の定義ファイル（CC BY-SA）をコピーしないことで、アプリに一切のライセンス制約が残らない

> NAWLの日本語定義ファイル自体もオンラインで配布されているが、それらは CC BY-SA。使うと「表示・継承」義務がアプリに付いてくる。訳を自作することでこれを回避している。

## 4. アプリの機能

### 学習モード（3種）
1. **フラッシュカード** — 英単語の下の赤い「暗記シート」をタップすると意味が現れる。「覚えた／あやふや」で仕分け
2. **4択クイズ** — 英単語の意味を素早く選ぶ
3. **単語帳を見る** — 一覧表示。赤シートで意味を隠して確認、1語ずつタップで一時的にめくれる

### 仕分け（軽いSRS）
- あやふやな語ほど優先的に再出題される（Leitner方式のボックス管理）
- 「覚えた」でボックスが上がり、「あやふや」「クイズ誤答」で下がる
- 定着（最上位ボックス到達）した語は出にくくなる

### レベル選択（英検4級〜1級）
- ホーム／単語帳の上部に横スクロールのレベルチップ（すべて・4級・3級・準2級・2級・準1級・1級）
- 各チップに語数を表示。選んだレベルだけを出題・表示
- 「単語 / 熟語 / すべて」の種別切り替えと組み合わせ可能

### 進捗表示
- ホームに定着率の円グラフ（定着 / 学習中 / 未学習）
- 連続学習日数（ストリーク）
- 進捗は端末に保存され、レベルをまたいでも語ごとに保持される

### デザインの考え方
- 日本の受験生になじみのある「赤シート（暗記シート）」を中心の操作に据えた
- 配色は「緑の下敷き × 紙のカード × 赤シート」。落ち着いた学習向けの雰囲気で、子供っぽくなりすぎないように

## 6. スマホでの使い方

1. `tango.html` を子供のスマホに送る（AirDrop / メール / クラウド経由など）
2. スマホのブラウザで開く
3. 共有メニューから「**ホーム画面に追加**」→ アプリのアイコンになる
4. 学習の進捗・連続日数は端末側に保存される

> Claudeのプレビュー画面上では進捗が保存されない（毎回リセットされる）が、実機のブラウザで開けば保存される。

## 7. 語を増やす方法

ファイル冒頭の `const DATA=[ ... ]` に1行足すだけ。

```javascript
{t:'word', lv:'2', en:'trait', pos:'名', ja:'特性、特徴'},
```

各フィールドの意味:

- `t` … `'word'`（単語）か `'idiom'`（熟語）
- `lv` … `'4' / '3' / 'p2' / '2' / 'p1' / '1'`（4級 / 3級 / 準2級 / 2級 / 準1級 / 1級）
- `en` … 英語
- `pos` … 品詞（名 / 動 / 形 / 副 / 熟）
- `ja` … 日本語の意味

この形式なら何百語でも同じように動く。
